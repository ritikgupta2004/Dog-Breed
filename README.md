## 📌 Overview

This project implements an end-to-end multi-class image classification
system to identify dog breeds using TensorFlow and transfer learning
(MobileNetV2).

------------------------------------------------------------------------

## 🎯 Problem Statement

Predict the breed of a dog from an input image along with confidence
scores.

------------------------------------------------------------------------

## 📂 Dataset

Kaggle Dog Breed Identification dataset:
https://www.kaggle.com/c/dog-breed-identification/data

------------------------------------------------------------------------

## 📄 CSV Dataset Description

  Column   Description
  -------- -------------------------------
  id       Image filename (without .jpg)
  breed    Dog breed label

Example: id,breed 001513dfcb2ffafc82cccf4d8bbaba97,golden_retriever

------------------------------------------------------------------------

## ⚙️ Tech Stack

Python, TensorFlow, TensorFlow Hub, NumPy, Pandas, Matplotlib

------------------------------------------------------------------------

## 🔄 Workflow

1.  Data Loading
2.  Preprocessing
3.  Label Encoding
4.  Model Training
5.  Evaluation
6.  Prediction

------------------------------------------------------------------------

## 🧠 Model

MobileNetV2 + Dense (Softmax)
